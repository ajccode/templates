Bem vindo ao Template 0

Esse template é um template de uma série de sites feitos por Alexandre José da cruz gitHub: @alexandrejc para mostrar habilidades aprêndidas no curso de web designer. O curso de web designer teve duração de 2 anos e com o que aprendi foi apenas o start para grandes desenvolvimentos nas áreas de web design e desenvolvimento front-end. As técnologias desenvolvidas no curso foram:
  1. Html5 - Corpo do site
  2. CSS3 - Estilo do site
  3. Javascript - Interatividade do site
  4. PHP - Desenvoldvimento do lado do servidor
  5. Mysql - Banco de dados para armazenamento
  6. Photoshop - Edição de imagens e banners animados
  7. Illustrator - Criação de botões e afins - autodidata
